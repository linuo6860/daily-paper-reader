## 摘要
准确评估 HER2 表达水平对于乳腺癌评估和靶向治疗方案的选择至关重要。然而，标准的多步免疫组化（IHC）染色资源密集、昂贵且耗时，且在许多地区往往无法获得。因此，直接从 H&E 切片预测 HER2 水平已成为一种潜在的替代方案。研究表明，利用 H&E 图像生成的虚拟 IHC 图像进行自动 HER2 评分是有效的。然而，像素级虚拟染色方法计算成本高，且容易产生可能传播诊断误差的重建伪影。为了解决这些局限性，我们提出了潜空间引导双流网络（LGD-Net），这是一个采用跨模态特征幻觉而非显式像素级图像生成的创新框架。LGD-Net 在训练期间由教师 IHC 编码器引导，学习将形态学 H&E 特征直接映射到分子潜空间。为了确保幻觉特征捕捉到临床相关的表型，我们通过轻量级辅助正则化任务，利用任务特定的领域知识（特别是细胞核分布和细胞膜染色强度）对模型训练进行显式正则化。在公开 BCI 数据集上的广泛实验表明，LGD-Net 达到了最先进的性能，显著优于基准方法，同时能够利用单模态 H&E 输入实现高效推理。

## Abstract
It is a critical task to evalaute HER2 expression level accurately for breast cancer evaluation and targeted treatment therapy selection. However, the standard multi-step Immunohistochemistry (IHC) staining is resource-intensive, expensive, and time-consuming, which is also often unavailable in many areas. Consequently, predicting HER2 levels directly from H&E slides has emerged as a potential alternative solution. It has been shown to be effective to use virtual IHC images from H&E images for automatic HER2 scoring. However, the pixel-level virtual staining methods are computationally expensive and prone to reconstruction artifacts that can propagate diagnostic errors. To address these limitations, we propose the Latent-Guided Dual-Stream Network (LGD-Net), a novel framework that employes cross-modal feature hallucination instead of explicit pixel-level image generation. LGD-Net learns to map morphological H&E features directly to the molecular latent space, guided by a teacher IHC encoder during training. To ensure the hallucinated features capture clinically relevant phenotypes, we explicitly regularize the model training with task-specific domain knowledge, specifically nuclei distribution and membrane staining intensity, via lightweight auxiliary regularization tasks. Extensive experiments on the public BCI dataset demonstrate that LGD-Net achieves state-of-the-art performance, significantly outperforming baseline methods while enabling efficient inference using single-modality H&E inputs.

### 1. 核心问题与研究动机
*   **核心问题**：如何高效、准确地直接从 H&E（苏木精-伊红）染色切片中预测乳腺癌的 HER2 表达水平（0, 1+, 2+, 3+）。

*   **动机**：现有的“虚拟染色”方法（如 GAN）试图将 H&E 转换为 IHC 图像，但存在两个主要缺陷：
    1.  **计算开销大**：像素级生成需要复杂的解码器。
    2.  **伪影风险**：生成模型可能产生与诊断无关的纹理或错误的染色伪影，导致误诊。
*   **目标**：提出一种无需显式图像生成、仅在特征层面进行“幻觉”映射的方法，以提高效率和诊断可靠性。

### 2. 方法论
LGD-Net 的核心思想是**跨模态特征幻觉（Feature Hallucination）**，即在潜空间中将 H&E 的形态学特征映射为 IHC 的分子语义特征。
*   **双流编码器（Teacher-Student 架构）**：
    *   **教师编码器（ET）**：在训练阶段处理真实的 IHC 图像，提取分子特征潜变量 $z_{IHC}$。
    *   **学生编码器（ES）**：处理 H&E 图像，提取形态学特征 $z_{HE}$。
*   **特征幻觉模块（M）**：学习一个非线性映射，将 $z_{HE}$ 转换为“幻觉”出的 IHC 特征 $\hat{z}_{IHC}$。
*   **领域知识正则化（关键创新）**：为了确保幻觉特征具有生物学意义，引入两个轻量级辅助解码器：
    *   **细胞核密度正则化**：预测细胞核分布（从 H&E 的苏木精通道提取）。
    *   **细胞膜强度正则化**：预测细胞膜染色完整性（从 IHC 的 DAB 通道提取），这是 HER2 评分的关键临床指标。
*   **动态融合与分类**：使用基于注意力的融合机制整合 $z_{HE}$ 和 $\hat{z}_{IHC}$，最后通过分类器输出评分。
*   **损失函数**：结合了分类交叉熵（$L_{cls}$）、特征蒸馏损失（$L_{dist}$，余弦相似度）以及生物学正则化损失（$L_{bio}$，包括 MSE 和 Dice Loss）。

*   **数据集**：使用公开的 **BCI 数据集**，包含 4,873 对严格配准的 H&E-IHC 图像块（1024×1024 像素）。
*   **实验划分**：官方训练集 3,896 张，测试集 977 张。
*   **对比基准（Benchmarks）**：

*   **评估指标**：准确率（Acc）、Macro-F1 分数、Cohen’s Kappa 系数。

*   **软件框架**：PyTorch。
*   **训练细节**：使用 AdamW 优化器，初始学习率 1e-4，采用余弦退火策略，训练 50 个 epoch。

### 5. 实验数量与充分性
*   **实验规模**：
    *   进行了主实验对比（6 种不同方法/配置）。
    *   **消融实验**：设计了 6 组变体（Variant A-F），分别验证了特征幻觉、注意力融合、细胞核正则化、细胞膜正则化对性能的贡献。
*   **充分性评价**：实验设计较为客观，涵盖了从单模态到最先进双模态方法的对比。消融实验清晰地展示了领域知识约束如何逐步提升模型在 HER2 评分（尤其是区分 1+ 和 2+ 等模糊类别）上的表现。

### 6. 主要结论与发现
*   **性能领先**：LGD-Net 在测试集上达到了 **95.60% 的准确率**，显著超过了仅使用 H&E 的基准（82.29%）和之前的 SOTA 方法（94.37%）。
*   **特征幻觉的有效性**：证明了在潜空间进行跨模态映射比在像素空间生成图像更有效，且能避免生成伪影。
*   **领域知识的重要性**：细胞膜强度正则化对性能的提升略高于细胞核正则化，这符合 HER2 评分主要依据细胞膜染色完整性的临床逻辑。
*   **推理高效性**：推理时只需 H&E 输入，且无需运行复杂的图像解码器，适合大规模临床筛查。

### 7. 优点
*   **架构创新**：通过“特征幻觉”替代“虚拟染色”，解决了生成模型计算量大和伪影干扰诊断的问题。
*   **临床可解释性**：通过辅助任务强行让模型学习细胞核分布和细胞膜强度，使黑盒模型具备了符合病理学逻辑的约束。
*   **端到端优化**：训练过程整合了蒸馏、正则化和分类，推理过程精简。

### 8. 不足与局限
*   **数据集单一**：实验仅在 BCI 一个公开数据集上完成，缺乏在多中心、不同扫描仪来源数据集上的泛化性验证。
*   **依赖配准数据**：训练阶段仍需要严格配准的 H&E-IHC 图像对，这在实际临床数据收集中心具有挑战性。
*   **领域偏移风险**：虽然避免了像素伪影，但如果 H&E 图像质量极差，特征层面的“幻觉”映射仍可能产生偏差，文中未深入探讨极端样本下的鲁棒性。
